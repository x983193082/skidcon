## ⚠️ 风险和注意事项

1. **CrewAI 版本兼容性**: CrewAI 更新频繁，API 可能变化，需关注版本兼容
2. **LLM API 成本**: 渗透测试任务可能产生大量 Token 调用，需控制成本
3. **法律合规**: 确保工具仅用于授权测试，添加免责声明和使用限制
4. **安全风险**: 工具本身包含攻击性代码，需做好访问控制
5. **工具依赖**: Nmap、SQLMap 等工具在不同系统上安装方式不同
