验证漏洞存在方法
    - 检查WordPress网站是否安装了Salon booking system插件且版本≤9.8
    - 尝试通过未授权访问删除任意文件

漏洞利用方法
    - 该漏洞是由于插件在删除上传文件时未正确验证文件路径，导致未授权攻击者可以删除任意文件，包括wp-config.php文件
    - 攻击者可以利用此漏洞删除关键系统文件，可能导致站点被接管和远程代码执行
    - 由于JSON数据中未提供具体的PoC代码，需要进一步研究漏洞利用细节或等待公开的PoC发布
