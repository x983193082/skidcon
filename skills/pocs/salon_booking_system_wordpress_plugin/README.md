# Salon Booking System WordPress Plugin PoCs

该目录包含 Salon Booking System WordPress 插件的漏洞PoC技能文件。

## 可用PoC技能

| 漏洞ID | 文件路径 | 描述 |
|--------|----------|------|
| CVE-2024-4442 | [cve_2024_4442.md](./cve_2024_4442.md) | Salon booking system插件任意文件删除漏洞（版本≤9.8） |

## 使用说明

每个PoC技能文件都遵循以下格式：
- `name`: 技能名称，格式为 `poc-[软件名]-[cve_id]`
- `description`: 技能描述，包含使用前提条件
- 验证漏洞存在方法
- 漏洞利用方法
