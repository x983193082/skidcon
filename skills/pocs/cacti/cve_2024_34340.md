验证漏洞存在方法
    - 检查Cacti版本是否小于1.2.27
    - 检查系统是否运行PHP版本<5.5.0
    - 检查密码验证逻辑是否使用`compat_password_verify`函数

漏洞利用方法
    - 这是一个类型混淆漏洞，在Cacti的`compat_password_verify`函数中，当使用MD5哈希时，密码验证使用松散比较（==）而不是严格比较（===）
    - 攻击者可以利用PHP的类型混淆特性，通过精心构造的MD5哈希值绕过密码验证
    - 由于`$md5 == $hash`的松散比较，某些特定的MD5哈希值（如以"0e"开头的科学计数法表示的数字）可能会被误判为相等
    
PoC示例：
```php
// 漏洞存在于lib/functions.php中的compat_password_verify函数
function compat_password_verify($password, $hash) {
    if (function_exists('password_verify')) {
        return password_verify($password, $hash);
    } else {
        $md5 = md5($password);
        // 漏洞点：使用松散比较 == 而不是严格比较 ===
        return $md5 == $hash;
    }
}

// 攻击者可以尝试使用以下类型的密码进行暴力破解：
// 寻找满足 md5(password) == "0e123456789012345678901234567890" 的密码
// 因为PHP中 "0e123..." == 0 会返回true

// 示例攻击向量：
// 1. 尝试使用已知的"魔术哈希"进行密码绕过
// 2. 利用PHP类型混淆特性构造特定的MD5哈希
```

注意：由于这是一个认证绕过漏洞，成功利用后攻击者可以无需有效凭证访问Cacti系统。
