# Nmap Scan Skill
## Use when
- 需要发现目标主机的开放端口时
- 需要识别目标服务版本时
- 渗透测试信息收集阶段
- 需要进行网络资产探测时
## What it does
提供完整的 nmap 扫描能力，包括：
- 快速扫描常用端口
- 完整端口扫描
- 服务版本检测
- 操作系统指纹识别
- 漏洞脚本扫描
## Instructions
### 扫描模式选择
1. **快速扫描 (Quick Scan)**
   - 适用：初步信息收集，时间有限
   - 扫描端口：21,22,23,25,53,80,110,143,443,445,3306,3389,5432,8080,8443
   - 使用工具：`quick_nmap_scan`
2. **标准扫描 (Standard Scan)**
   - 适用：需要较完整信息
   - 使用工具：`nmap_port_scanner`
   - 参数：
     - `scan_type`: -sV (版本检测), -sS (SYN扫描), -sT (TCP扫描)
     - `ports`: 端口范围，如 "1-1000" 或 "80,443,8080"
3. **全面扫描 (Full Scan)**
   - 适用：深度渗透测试
   - 使用工具：`full_nmap_scan`
   - 包含：操作系统检测 (-O) + 漏洞脚本 (--script=vuln)
### 执行流程
1. 验证目标格式（IP 或域名）
2. 选择扫描模式
3. 执行扫描
4. 解析结果，提取关键信息
5. 报告发现的端口和服务
### 示例
- User: 扫描 192.168.1.1 的常用端口
Agent: 使用 quick_nmap_scan 工具
- User: 扫描 192.168.1.1 的 1-1000 端口
Agent: 使用 nmap_port_scanner，设置 ports="1-1000"
- User: 进行全面扫描
Agent: 使用 full_nmap_scan
### 注意事项
- 扫描前确认授权
- 全面扫描可能耗时较长（配置 timeout）
- 结果需要人工验证
